Detailed Description

ValPacket represents the packets received and sent by VAL.

This class contains the VAL Header and the NDN packet.
The objects have 3 types of state.
NOT_SET - 0
DATA_SET - 1
INTEREST_SET - 2
Every object is created in NOT_SET state.
Upon the usage of function setInterest or setData the state changes. The state can only be changed once.
