calculates the timer duration based on a distance

it can be used to calculate times for Interests or Data packets
times for distances to a geo-area
times for distances between nodes
follows the less distance less time approach

Parameters
distthe distance, it can be between points or between point and geo-area
hopcnumber of hops that a packet still has - default value 0
toAreaa boolean that tells the function that the distance is in ralation to a geo-area if set to True
isDataa boolean that tells the function that the timer is for a Data packet if set to True
